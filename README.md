# Coral-Settlement-Northern-GBR
Thesis Analysis

These files are used for the thesis by Monique J White, James Cook University.
Title: Coral recruitment

File: Data_Analysis_Thesis is the main working r markdown
for the analysis for the thesis

Coralsettlement_nGBR_CS.xlsx is the original data file provided from Andrew Hoey.

Benthic surveys are provided in Cross-shelf benthic surveys_2013-25_nGBR.csv.
I added some columns in excel which calculated the sum of:
- Macroalgae
- soft coral
- Acroporidae (Acropora, Isopora, Montipora)
- Pocilloporidae ()
- Poritidae (Porites)
- Other
- CCA
- Sand/rubble
- Dead coral
